# Activate Backend

In `FellowshipFinder/Backend`:

- `chmod +x setup_venv.sh` \\
- `./setup_venv.sh` \\
- `source .venv/bin/activate` \\

You will need an Open AI API key or a local GPU that can run mistral-7b. Make sure to export the OpenAI API key.
